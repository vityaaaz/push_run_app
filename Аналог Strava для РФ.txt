1. Web-версия с мобильным адаптивом  — это разумно для MVP, так как:  
- Быстрее в разработке (не нужно отдельно делать iOS/Android).  
- Дешевле (один код вместо двух платформ).  
- Легче тестировать и обновлять.  


   2. Корректировка технической реализации   
 #  2.1. Фронтенд (Web + Mobile Adaptive)   
 Технологии:   
-  Next.js  (React-фреймворк с SSR для SEO + мобильного адаптива).  
-  PWA (Progressive Web App)  → можно установить на смартфон как приложение.  
-  Tailwind CSS  или  MUI  – для быстрого responsive-дизайна.  
- Nuxt3 (Vue3 based фреймворк с SSR для SEO)
- Vue3 + Ionic для мобильной платформы. (Есть возможность писать нативные модули)
- Tailwind CSS + schadcn-vue для web и мобилки как отдельный пакет


 Ключевые страницы:   
-  Трекинг  (карта + кнопка старта/стопа).  
-  Профиль  (статистика, друзья).  
-  Лента активностей  (как в Strava).  
-  Челленджи  (список текущих).  


 Мобильный UX:   
- Горизонтальный свайп между разделами.  
- Крупные кнопки для бега/велосипеда.  
- Оффлайн-работа (кеширование через Service Workers).  


 #  2.2. Бэкенд (Упрощённый вариант)   
 API:   
-  Python (FastAPI)  или  Node.js (NestJS)  – для основного функционала.  
-  PostgreSQL  – хранение пользователей и активностей.  
-  Redis  – кеширование рейтингов.  


 Эндпоинты (REST):   
- `POST /api/track` – запись трека.  
- `GET /api/feed` – лента друзей.  
- `POST /api/challenge` – создание челленджа.  




 #  2.3. Интеграции   
-  Яндекс.Карты  (для отображения маршрутов).  
-  ВКонтакте API  – авторизация и шеринг.  
-  Telegram Bot  – уведомления о новых событиях.  










   3. MVP-функционал (первая версия)   
1.  Трекинг маршрутов  (GPS через браузер).  
2.  Базовая статистика  (дистанция, время, скорость).  
3.  Соц. фичи:   
   - Подписки на других пользователей.  
   - Лента активностей.  
4.  Челленджи  (например, «Пробеги 50 км за месяц»).  
   4. Дизайн (русскоязычный UI)   
 Стиль:   
- Минимализм (как Strava, но с акцентом на зелёный/синий).  
- Крупные элементы для мобильных.  
   5. Инфраструктура   
-  Хостинг:  VK Cloud (дешевле AWS в РФ).  
-  Домен:  найди свободный еще епта (≈ 500–1000 ₽/год).  
-  CI/CD:  GitHub Actions (автодеплой при пуше в main).
